 Five vulnerabilities have been discovered in the Joomla content management system, including CVE-2024-21722, CVE-2024-21723, CVE-2024-21724, CVE-2024-21725, and CVE-2024-21726. These vulnerabilities could be exploited to execute arbitrary code on vulnerable websites. Joomla has issued fixes in versions 5.0.3 and 4.4.3. CVE-2024-21725 is the most severe vulnerability with a high exploitation probability. Stefan Schiller warns that CVE-2024-21726 could be leveraged to achieve remote code execution through user interaction. All Joomla users should update to the latest version to mitigate the risks associated with these vulnerabilities.
